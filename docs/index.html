Webcast Internal Tool

Webcast is a self-hosted live streaming control panel used internally by a single church organization to automate YouTube Live broadcasts of Sunday meetings.

What it does

  • Schedules and automatically starts YouTube Live broadcasts at configured times
  • Controls a PTZ camera via a web-based control panel
  • Monitors stream health and handles recovery from failures
  • Sends attendance summary emails after each broadcast
  • Manages recurring weekly schedules and one-off events

YouTube API usage

Webcast uses the YouTube Data API v3 to create and manage live broadcasts on behalf of the church's YouTube channels. OAuth authorization is performed once per channel and refresh tokens are stored locally on the church's own server. No data is shared with any third party.

Internal use only. This application is not available to the public and has no external users. It is operated exclusively by volunteer administrators within a single church organization.
