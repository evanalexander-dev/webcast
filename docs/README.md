This folder contains the GitHub Pages website for the Webcast project, served at:

- **Homepage:** https://evanalexander-dev.github.io/webcast/
- **Privacy policy:** https://evanalexander-dev.github.io/webcast/privacy.html

These pages exist to satisfy Google's OAuth consent screen requirements for the YouTube Data API integration. They describe how the application uses YouTube API data and provide the required privacy policy URL.

To enable GitHub Pages for this repo: **Settings → Pages → Source: Deploy from branch → branch: main, folder: /docs**.
